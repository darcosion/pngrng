# pngrng
Programme qui génère un png dont les pixels sont de la pure randomness. Le mode noir & blanc n'est pas sans rappeler "la neige" des vieilles télévisions.


## installation

```
make
chmod +x pngrng
```

## utilisation

```
./pngrng
```
